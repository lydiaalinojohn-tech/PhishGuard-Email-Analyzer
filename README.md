# PhishGuard Email Analyzer

## Overview

PhishGuard is a Python-based cybersecurity tool designed to identify phishing indicators in email content.

## Features

- Suspicious Keyword Detection
- URL Analysis
- Risk Classification
- Security Recommendations

## Technologies Used

- Python 3
- Regular Expressions
- Cybersecurity Fundamentals

## Sample Output

========== SECURITY REPORT ==========

Suspicious Links Found: 1

Risk Level: HIGH

Recommendation:
Do NOT click links or provide credentials.

## Skills Demonstrated

- Email Security
- Phishing Detection
- Social Engineering Awareness
- Risk Assessment
- Python Programming
